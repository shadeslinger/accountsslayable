# Accounts Slayable — Image Style Guide

This file is the single source of truth for AI image generation on the site.
Every image generated by the bot, by Adam, or by any contributor should follow
this style so the site feels visually coherent.

The `scripts/generate-image.mjs` script reads this file at generation time and
prepends the **STYLE PREAMBLE** block below to every prompt.

---

## STYLE PREAMBLE

> Flat 2D vector illustration in a modern editorial style. Soft sage green
> (#87A96B), warm cream (#F7F5F3), coral pink (#FF6B6B), and deep charcoal
> (#2D2D2D) color palette. Clean geometric shapes, gentle shadows, subtle
> gradients. Playful but not childish, friendly but not corporate. No human
> faces. No text or lettering in the image. No brand logos. No realistic
> photography. Composition centered with breathing room around the subject.
> Background is cream or soft sage with abstract geometric accents (circles,
> rounded squares, curves). Subject-focused, clean, uncluttered.

(Do not modify the preamble without discussing with Adam — consistency is the
whole point.)

---

## Visual vocabulary

These are the elements that repeat across the brand. When prompting for a
specific illustration, try to use 1–2 of these as "anchor objects":

- **Spreadsheets, charts, bar graphs, line graphs** — but abstract and clean, not Excel screenshots
- **Coins, dollar bills, stacked cash** — stylized, not realistic
- **Credit cards** — stylized, no real brand logos
- **Calendars, clocks, timers** — for time-based concepts (quarterly taxes, deadlines)
- **Plants, leaves, growing shoots** — for growth, saving, progress
- **Envelopes, receipts, mail, paperwork** — for invoicing, admin
- **Laptops, phones, tablets** — for AI/tech/fintech content, screen showing abstract UI
- **Simple isometric buildings** — for homebuying, real estate
- **Abstract geometric shapes** — circles, wavy lines, rounded rectangles, used as background decoration
- **Arrows, checkmarks, badges** — for process/progress
- **Suitcases, passports, airplane silhouettes** — for travel rewards

Avoid: human faces, realistic renderings, stock-photo-style scenes, finance-bro
imagery (watches, cigars, skyscrapers, bull/bear), religious or political imagery.

## Image types and their prompts

### Blog post hero (1024 x 576, landscape 16:9)

**Template:**
```
Blog post hero illustration about [TOPIC]. Central subject: [1–2 anchor
objects from vocabulary above]. Mood: [punchy/calm/curious/playful]. Background:
soft cream with abstract geometric accents in sage and coral.
```

**Example (credit card rewards guide):**
```
Blog post hero illustration about credit card rewards for beginners. Central
subject: a stylized credit card with coins and small stars floating above it,
with a subtle upward arrow in coral. Mood: playful and inviting. Background:
soft cream with abstract geometric accents in sage and coral.
```

### Pillar illustration (1024 x 1024, square)

Each of the six pillars has one hero illustration. These are generated once
and committed to `/public/images/pillars/`. They are the visual identity of
the pillar.

**Template:**
```
Hero illustration for a personal finance website pillar called "[PILLAR NAME]".
Central symbol: [pillar-specific object]. Background: cream with sage or coral
geometric accents. Square composition, centered.
```

**Pillar-specific anchors:**
- Personal Finance → a plant growing from a stack of coins or a simple bar chart with upward arrow
- Points & Rewards → a stylized credit card with stars and a paper airplane
- Freelance & 1099 → a clipboard with invoice lines and a small coffee cup
- AI + Fintech → a laptop or phone showing an abstract chart UI, with a small spark/lightning bolt
- Side Hustles & Biz → an isometric shop or toolkit with multiple small icons stacked
- Money Mindset → a head-silhouette made of abstract shapes with thought-bubbles (no actual face)

### Decorative accent (512 x 512, square)

Small illustrations for section headers, quote cards, or inline breaks.

**Template:**
```
Small decorative illustration of [single object]. Flat vector, minimal,
centered on cream background. For use as a section accent.
```

### Product screenshot placeholder (1200 x 800, 3:2)

For the 1099 product page and future product pages — an illustration of a
stylized spreadsheet or template, not a real screenshot.

**Template:**
```
Illustration of an abstract spreadsheet interface showing [PRODUCT]. Stylized
cells, charts, and numbers in sage and coral. Clean, modern, editorial style.
No text legible, just suggestive of a well-organized spreadsheet.
```

## Prompt engineering tips

- **Be specific about composition.** "Centered," "left-aligned," "top-down view" all matter more than you'd think.
- **Avoid "professional" as a descriptor.** It pushes the model toward stock photography.
- **Mention the palette by name AND hex code.** "Soft sage green #87A96B" works better than either alone.
- **Negative prompts aren't supported in Flux schnell.** Bake all "no X" constraints into the positive prompt itself ("no human faces, no text, no logos").
- **Iterate at low cost.** Flux schnell is $0.003/image — generate 5 variants, pick the best, commit that one.

---

## Output conventions

- File format: **WebP** if possible, **PNG** otherwise. The script converts from fal.ai's JPEG response to WebP using nothing — actually we just save the raw download.
- File location: `/public/images/<category>/<slug>.webp`
- Filename: kebab-case, no spaces, no uppercase, no version suffixes (`.v2`, `.final`, etc. — use git for that)
- Always commit with the prompt used, either in a comment or in this file's CHANGELOG section below.
